## 🎯 О проекте

Тренажёр разработан для обучения операторов БПЛА навыкам экологического мониторинга лесных территорий. Имитирует реальную работу дрона-эколога с обзором 360° и физикой полёта.

### 📋 Задачи миссии:
- 🔥 Обнаружение лесных пожаров
- 🪓 Выявление незаконных вырубок
- 🏭 Мониторинг промышленных загрязнений
- 🦌 Борьба с браконьерством
- 💧 Обнаружение разливов нефти

> ⚠️ **Главное правило**: столкновение с деревом = провал миссии!

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| `W` / `↑` | Движение вперёд |
| `S` / `↓` | Движение назад |
| `A` / `←` | Движение влево |
| `D` / `→` | Движение вправо |
| `Q` / `Shift` | Набор высоты |
| `E` / `Ctrl` | Снижение |
| 🖱️ **Мышь (зажать)** | Обзор 360° |

## ✨ Возможности

- 🎥 **Обзор 360°** — полный контроль пространства
- 🚁 **Детализированная 3D-модель дрона** — карбоновый корпус, 4 мотора, вращающиеся пропеллеры, LED-индикация, подвесная камера
- 🌲 **Процедурная генерация леса** — каждый запуск уникален
- 🔍 **Система сканирования** — приближайтесь к проблемам для их обнаружения
- 🔋 **Мониторинг батареи** — планируйте маршрут экономно
- 📡 **Телеметрия в реальном времени** — высота, скорость, сигнал
- 🎵 **Звуковые эффекты** — WebAudio API
- 💥 **Физика столкновений** — реалистичная обработка падений
- 🏆 **Система оценки** — за найденные проблемы начисляются очки
- 📱 **Адаптивный дизайн** — работает на мобильных устройствах
